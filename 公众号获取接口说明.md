# 一、通过公众号名称/微信Id/链接获取公众号当天发文情况
该接口获取的是实时数据，非数据库陈旧数据。
返回当天发文数据，最多获取账号20次发文（注：99%的账号每天只发1次，个别新闻媒体账号可以多发，一般不超过20次，每次1到8篇），本接口无需翻页。
注：如果账号当天还未发文，也需要扣费。
由于腾讯规则调整2024年12月1号起 历史接口返回的文章链接均为短链接
## 状态码	说明
{"message":"Internal Server Error"}	网络错误，请重试1~3次
0	成功
-1	QPS超过上限，不得高于5次/秒，请5秒后再试！
100	缺少参数 biz,name,url 三者必须至少有一个
101	文章被删除或公众号已被封或迁移
104	短连接文章已被删除或文章被迁移
105	没找到该名称的公众号，建议您采用biz或者url作为参数获取！
111	请求频繁，请稍后再试。
112	请求失败，请稍后再试。
113	鉴权失败，请稍后再试。
400	短链接转化失败，建议先调用 短链接转长链接 转化为长链接后继续调用
2005	系统错误，请重新尝试。
10002	key或附加码不正确
20001	金额不足，请充值
20002	请输入正确的微信链接
20003	文章链接有误，请检查文章链接url中的&是否已经编码为%26|50000|内部服务器错误
## 返回结果
字段	说明
['data'][i]['position']	发文位置（头条
['data'][i]['url']	文章链接
['data'][i]['post_time']	发文时间戳
['data'][i]['post_time_str']	发文时间文本显示方式
['data'][i]['cover_url']	文章封面
['data'][i]['original']	1:原创 0:未声明原创 2:转载
['data'][i]['item_show_type']	0:图文 5:纯视频 7:纯音乐 8:纯图片 10:纯文字 11:转载文章
['data'][i]['digest']	摘要
['data'][i]['title']	文章标题
['data'][i]['pre_post_time']	定时发文时间戳
['data'][i]['msg_status']	2：正常； 7：已被删除； 6：文章因违规发送失败 104 审核中 105 正在发送中
['data'][i]['msg_fail_reason']	文章被删除或者违规的原因
['data'][i]['send_to_fans_num']	收到群发消息的粉丝数量（发布的文章 或者 发送失败的文章此参数为0）
['data'][i]['is_deleted']	0：正常； 1：已被删除
['data'][i]['types']	类型 9 ：群发，有通知，1天发送1次； 类型 1：发布，无通知，无次数限制
['total_num']	发文总次数（群发+发布）
['total_page']	总页数（每页返回5次/天的内容；屏蔽搜索账号每页返回10次/天的内容，但无法获取总页数，价格贵0.06）
['publish_count']	发布总次数
['masssend_count']	群发总次数
['now_page']	当前页（注：1次发文最多包含8篇文章；每页返回5次发文，屏蔽搜索账号每页返回10次发文）
['now_page_articles_num']	当前页文章篇数
['cost_money']	消费金额
['remain_money']	剩余金额
['mp_nickname']	公众号名称(仅name作为参数名调用时显示)
['mp_wxid']	公众号wxid(仅name作为参数名调用时显示)
['mp_ghid']	公众号原始id(仅name作为参数名调用时显示)
['head_img']	公众号头像(仅name作为参数名调用时显示)
## 请求参数
Header 参数
Content-Type string 可选
示例:application/json
Body 参数 application/json
json
biz string 可选 biz调用参数名biz(0.06元/次) 注意：biz、 url、 name 任选一个填入，如果都填，优先使用biz 然后 url 最后 name
url string 可选 微信文章链接（短链接0.08/次、长链接0.06/次）
name string 可选 微信id和公众号名称调用参数名name(0.08元/次)
key string 必需 极致了官网 key
verifycode string 可选 附加码，如设置了附加码verifycode，则此参数为必选，如未设置则为非必选
请求参数示例：
{
  "biz": "",
  "url": "https://mp.weixin.qq.com/s?__biz=MjM5MTM5NjUzNA==&mid=2652494556&idx=1&sn=4995d845ad2ef1205136936f65ae4adc&chksm=bd5b5d058a2cd4139bbd92c8cd23d52f65ef260eedf8cbc6d25a4ab0992f08d01da81#rd",
  "name": "",
  "key": "{{key}}",
  "verifycode": ""
}

# 二、获取文章正文 HTML
GET
https://www.dajiala.com/fbmain/monitor/v3/article_detail
## 状态码	说明
{"message":"Internal Server Error"}	网络错误，请重试1~3次
0	成功
101	文章被删除或违规或公众号已迁移
105,106	文章解析失败
107	解析失败，请重试
qps=5
## 返回结果
字段	说明
['code']	状态码
['cost_money']	消费金额
['remain_money']	所剩金额
['type']	9 群发的有通知 10002 发布无通知
['content']	纯文本格式正文
['content_multi_text']	适用于富文本编辑器的html格式
['title']	文章标题
['url']	文章长链接
['hashid']	文章唯一id
['mp_head_img']	公众号头像
['picture_page_info_list']	文章中图片列表
['cdn_url_1_1']	1:1的封面图
['nick_name']	公众号名字
['user_name']	公众号原始id
['idx']	文章位置，8篇里面第几篇
['msg_daily_idx']	比如人民日报一天可以发10次，这是第几次发送
['pubtime']	发文时间
['create_time']	文章创建（定时）时间
['biz']	公众号biz
['alias']	公众号wxid
['source_url']	文章原文链接 （如果没有则为空）
['signature']	公众号简介
['author']	文章作者
['desc']	文章摘要
['copyright_stat']	是否原创 （1 原创 0非原创 2转载）
['ip_wording']	ip地址
['item_show_type']	0:图文 5:纯视频 7:纯音乐 8:纯图片 10:纯文字 11:转载文章
['real_item_show_type']	0:图文 5:纯视频 7:纯音乐 8:纯图片 10:纯文字 11:转载文章
['video_page_infos']	文章中的视频列表
## 请求参数
Query | 参数 
url | string | 必需 | 微信文章链接 
key | string | 必需 | 极致了官网 key 
mode | string | 可选 | 1.带图片标签纯文本 2.纯文字+富文本格式
verifycode | string | 必需 | 附加码，如设置了附加码verifycode，则此参数为必选，如未设置则为非必选
